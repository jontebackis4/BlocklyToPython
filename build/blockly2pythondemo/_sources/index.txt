==========
Välkommen!
==========

På nästa sida, som länkas nedan, hittas exempel på några av de verktyg och moduler som kan användas för att bygga upp ett interaktivt läromedel. Förutom statisk text, kan videoklipp läggas in, och även programkod där användaren kan testköra koden direkt i webbläsaren *utan* att man först behöver ladda ned och installera någon programvara. Det finns stöd för ett flertal olika programmeringsspråk. Exempel på andra komponenter och moduler som kan användas för att bygga upp sitt interaktiva läromedel är:

* Flervalsfrågor
* Lucktester
* Kommentarer
* Kommentarsfält
* Kodlinsen - för att stegvis gå igenom programmeringskod
* Dra och släpp
* Tidsbegränsade prov
* Programmeringsproblem som ungdomarna får lösa
